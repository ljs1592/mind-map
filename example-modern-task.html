🎨 现代化思维导图 + 任务管理

基于 simple-mind-map 的现代化主题和任务勾选框功能演示

✨ 功能说明

🎯 任务勾选框功能

  • 点击节点前的勾选框切换任务状态
  • 叶子节点显示方形勾选框
  • 父节点显示圆形进度指示器
  • 进度根据子节点自动计算
  • 点击父节点可批量操作子任务

🎨 Modern 主题特性

  • 类 React Flow 简约风格
  • Tailwind CSS 配色方案
  • 柔和的圆角和阴影
  • 优雅的曲线连接线
  • 现代系统字体
Modern

⌨️ 快捷键

  • Tab - 添加子节点
  • Enter - 添加同级节点
  • Delete - 删除节点
  • Ctrl+C/V - 复制/粘贴
  • Ctrl+Z/Y - 撤销/重做
键盘导航

📝 使用方法

  • 双击节点编辑文本内容
  • 拖拽节点调整结构
  • 鼠标滚轮缩放画布
  • 按住鼠标拖动平移视图
